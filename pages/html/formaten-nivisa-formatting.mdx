# **Formatên Nivîsa (Formatting)**

Paragrafek her gav li ser rêzek nû dest pê dike, û bi gelemperî bloka nivîsê ye.

Paragrafên HTML'ê bi nîşanka `<p>` tên diyar kirin.

Paragraf di navbera nîşankên `<p>` û `</p>` de tên nivîsandin.

```html copy showLineNumbers
<p>
  Tîp û lekên kurd ketin zinaran <br />
  Li hawîr bumba bû gullebaran <br />
  Êrîşdibin ser komên neyaran <br />
  De rabe ser xwe, kurê min rabe! <br />
  <br />
  -Seydayê Cigerxwîn
</p>
```

Hêmanên ku di nava paragrafan yan jî nivîsan de tên bikar anîn heye.

Bi wan re **Hêmanên Alîkar** tên gotin.

Em wan hêmanan lê binêrin û pênase bikin.

## **Hêmana `<br>`**

Hêmana `<br>` veqetandina rêzê diyar dike.

Heger hûn xêzek (xêzek nû) dixwazin, `<br>` dest pêkirina paragrafek nû dikare bîne.

**Nîşe:** `<br>` Hêmaneke Vala ye û nîşankê dawî nayê bikaranîn.

```html copy showLineNumbers
<p>Ev paragraf <br />mînakê rêzedersên <br />HTML'e.</p>
```

## **Hêmanên `<b>` û `<strong>`**

Hêmana `<b>` nivîsa stûr dike. Girîngiyek destnîşan nake.

```html copy showLineNumbers
<b>Ev nivîsekê stûr e.</b>
```

Hêmana `<strong>` nivîsê bi girîngiyek xurt diyar dike.

Naveroka hundur bi gelemperî bi stûr tê xuyang kirin.

```html copy showLineNumbers
<strong>Ev nivîsekê girîng û stûre!</strong>
```

## **Hêmanên `<i>` û `<em>`**

Hêmana `<i>` beşek nivîsê bi dengek an hestek alternatîf diyar dike.

Naveroka hundur bi gelemperî bi îtalîk tê xuyang kirin.

`<i>` bi gelemperî ji bo nîşankirina peyvên teknîkî, hevokek ji zimanek din, ramanek, hwd tê bikar anîn.

```html copy showLineNumbers
<p>Pergala Xebitandine - <i>Mînak: Windows, Linux, MacOS</i></p>
```

Hêmana `<em>` nivîsa balkêş diyar dike.

Naveroka hundur bi gelemperî bi îtalîk tê xuyang kirin.

```html copy showLineNumbers
<p>Bager îro fêrî HTML dibe. <em>Ji bo vê perwerdeyê pir kêyfxweşe!</em></p>
```

## **Hêmana `<small>`**

Hêmana `<small>` nivîsa piçûk diyar dike.

```html copy showLineNumbers
<small>Ev nivîsa hate piçûk kirinê ye.</small>
```

## **Hêmana `<mark>`**

Hêmana `<mark>` nivîsa ku divê were nîşankirin an jî piştronî kirin diyar dike.

```html copy showLineNumbers
<p><mark>Li pişt vê nivîsê</mark> hate ronî kirin.</p>
```

## **Hêmana `<del>`**

Hêmana `<del>` nivîsa ku ji belgeyek hatî jêbirin diyar dike.

Gerok bi gelemperî di nav nivîsa jêbirin de rêzek lê dixe.

```html copy showLineNumbers
<p>Mînak: <del>Ev gotin bi hêmana "del" ve hate nivîsandin.</del></p>
```

## **Hêmana `<ins>`**

Hêmana `<ins>` nivîsek ku di nav belgeyek de hatî danîn destnîşan dike.

Gerok bi gelemperî binê nivîsa lêxistî xêz dike.

```html copy showLineNumbers
<p>Mînak: <ins>Ev nivîsa bin xêz kiriye.</ins></p>
```

## **Hêmana `<sub>`**

Hêmana `<sub>` binnivîsê -nivîsa bin xistî- diyar dike.

Nivîsa binnivîsê nîv karakter li binê rêza normal xuya dike, û carinan bi tîpek piçûktir tê pêşkêş kirin.

Hêmana binavkirî dikare ji bo formulên kîmyewî, mîna H 2 O, were bikar anîn.

```html copy showLineNumbers
<p>Mînak: Ev nivîs <sub>hate berjêr kirin.</sub></p>
```

## **Hêmana `<sup>`**

Hêmana `<sup>` nivîsa sernivîsê diyar dike.

Nivîsa sernivîsê nîv karakter li jor rêza normal xuya dike, û carinan bi tîpek piçûktir tê pêşkêş kirin.

Nivîsara sernivîsê dikare ji bo jêrenotan were bikar anîn. Mînak: WWW \[1\]

```html copy showLineNumbers
<p>Ev nivîsa <sup>sernivîsê</sup> ye.</p>
```
