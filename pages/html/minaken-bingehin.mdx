# **Mînakên Bingehîn (Basic Examples)**

Di vê beşê de em ê hin mînakên bingehîn ên HTML'ê nîşan bikin.

Xem neke heger em hêmanên ku we hîn nebûne bikar bînin.

Niha tenê bala xwe bidin dersê û hewl bidin ku mantiqa wê fêr bibin.

Beşên pêşî de em bi kûrahî wan hêmanan fêr bibin.

## **Pelên HTML'ê**

Pelên HTML'ê nava kompîturê bi pêveka `.html` an jî `.htm` tên tomar kirin. Mînak: **navêpel.html**

Divê hemî pelên HTML'ê bi hêmana `<!DOCTYPE html> <html>` destpê dikin û bi hêmana `</html>` dawî dibin.

Qada rûpela HTML'ê ya ku naverok lê werin xuyang kirin bi `<body>` dest pê dike û bi `</body>` diqede.

    ```` html copy showLineNumbers filename="index.html"
    <!DOCTYPE html> <!--- Nîşankirina HTML5'ê --->
    <html> <!--- Pela HTML Destpêkir --->

    <body> <!--- Laşê Pelê / Naveroka pelê li vir tên nîşan kirin. --->

        <h1>Sernivîsê Destpêkê</h1>
        <p>Ev paragrafê min e.</p>

    </body> <!--- Laşê Pelê Qediya ye --->

    </html> <!--- Pela HTML5 Qediya ye --->
    ````

## **Nîşankirina `<!DOCTYPE html>`**

Nîşaneka `<!DOCTYPE html>` rûpela HTML5 temsîl dike, û ji gerokan re dibe alîkar ku rûpelên Web'ê rast nîşan bide.

Pêwiste ku ew tenê carekê, li serê rûpelê _(berî nîşanekên HTML'ê)_ xuya bibe.

## **Sernavên HTML'ê**

Sernavên HTML'ê ji nîşanka `<h1>` heta nîşanka `<h6>` tên diyar kirin.

Nîşanka `<h1>` sernavê girîng nîşan dike û sernava herî mezin ev e.

Nîşanka dawî ya `<h6>` jî sernava herî piçûk e.

**Minak:**

```html copy showLineNumbers
<h1>Sernavê Mezin Ev e</h1>
<h2>Rêzedersên HTML'ê</h2>
<h3>Mînakên Bingehîn</h3>
<h4>Pêşvebirina Nivîsbarîn</h4>
<h5>Pêşvebirên Kurd</h5>
<h6>PAWE</h6>
```

## **Paragrafên HTML'ê**

Paragrafên HTML'ê bi nîşanka `<p>` tên diyar kirin.

```html copy showLineNumbers
<p>Ev paragrafê HTML e. Li vir nivîs cih digre.</p>
```

## **Girêdankê HTML'ê**

Girêdankên HTML'ê bi nîşanka `<a>` tên diyar kirin.

Ev girêdankan rûpel an jî malperekê cûda rênîşan dike.

Navnîşana rûpelê di nava hêmana `href` tên nivîsandin.

Di nava nîşankên `<a>` û `</a>` yê sernivîsa vê girêdank tên nivîsandin.

```html copy showLineNumbers
<a href="https://pawe.dev">Pawe Pêşvebirin</a>
```

## **Wêneyên HTML'ê**

Di nava rûpelên HTML'ê wêneyan bi nîşankê `<img>` tên bi cih kirin.

Navnîşana wêneyê di nava taybetmendiya `src` tên nivîsandin.

Heger dixwazin di derbarê wêne agahîyekê bidin, di nava taybetmendiya `alt` bi nivîsin.

Dirêjahî û bilindahîya wêne bi taybetmendiyên `width` û `height` ve tên diyar kirin.

```html copy showLineNumbers
<img src="/pawe.png" alt="Rêzedersên HTML'ê" width="100" height="30" />
```
