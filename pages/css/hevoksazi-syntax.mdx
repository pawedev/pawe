# **Hevoksazî (Syntax)**

Hevoksazîya CSS, ji hilbijêrek û bloka danezanê pêk tê.

**Hilbijêr**, hêmana HTML'ê ya ku hûn dixwazin şêwaz bikin destnîşan dike.

Di bloka danezanê de yek an jî çend danezan hene ku bi **Xalebêhnok** `;` ve veqetandî ne.

Her danezan navek taybetmendiya CSS û nirxek heye, ku bi xalebêhnok ve veqetandî ye.

Gelek danezanên CSS'ê bi xalebêhnok têne veqetandin, û blokên danezanê bi **Kevaneyên Xemilî** `{ }` ve têne dorpêç kirin.

## **Mînaka Hevoksazîyê**

Di vê mînakê de hemî hêmanên `<p>` dê li navendê, bi rengek nivîsê ya sor bêne rêz kirin:

```css copy showLineNumbers
p {
  color: red;
  text-align: center;
}
```

**Pênase:**

- `p` : Di CSS'ê de hilbijêrek e. (ew hêmana `<p>` ya ku hûn dixwazin şêwaz bikin destnîşan dike).
- `color` : milkek e, û `red` nirxê vê milkê ye.
- `text-align` : milkek e, û `center` nirxê vê milkê ye.

**Çawa Xuya Dike?**

```html copy showLineNumbers
<style>
  p {
    color: red;
    text-align: center;
  }
</style>

<p>Rêzedersên CSS'ê Destpêkir!</p>
```
