# **Destpêkê**

CSS, ew ziman e ku em ji bo şêwazek rûpelên HTML'ê bikar tînin.

CSS diyar dike ku hêmanên HTML'ê çawa têne xuyang kirin.

Ev rêzeders dê CSS ji bingehîn heya pêşkeftî fêrî we bike.

## **CSS çi ye?**

**CSS**; tê wateya **Rûpelên Şêwazan Raxistî** *(Cascading Style Sheets)*.

CSS diyar dike ku hêmanên HTML'ê çawa li ser xuyanek, rûpel, an di medyayên din de têne xuyang kirin. **CSS gelek kar xilas dike.**

Ew dikare sêwirana çend rûpelên malperê bi yekcarî kontrol bike.

Kodên şêwazê di pelên CSS de têne hilanîn.

Pelên CSS'ê nava kompîturê de bi pêvekê `.css` tên tomar kirin. **Mînak: sewaz.css**

## **Çima CSS Bikar Bînin?**

CSS ji bo danasîna şêwazên ji bo rûpelên we yên webê, sêwirandin û guhertinên di pêşandanê de ji bo cîhazên cuda û mezinahiyên xuyanekê tê bikar anîn.

Bi pelek şêwazên CSS'ê, hûn dikarin bi guhertina pelek tenê dîmenê tevahiya malperek biguherînin.

CSS, ji bo afirandina malperên Web'ê **teqez** pêwiste!

Fêrbûn û bikaranîna CSS gelek hêsane.

### **Mînaka Kodên CSS'ê**

```css copy showLineNumbers
body {

}
```
